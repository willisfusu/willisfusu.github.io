    网站改版-05-APR-2020

    2020-04-07 | 技术 Gridea | 3 min read

    此文章发表于 625 天前，请注意文章时效

    英国的疫情不知道什么时候才能结束，原本在家应该是做点学术工作，处理处理数据，写写论文之类的。但是实在是没有心情去做这方面的工作，于是我只好找点事情来搞搞。

    最近看了不少让我眼前一亮的网站，于是就想着把自己的网页也改版一下。🎉

    因为我自己对于设计实在是没有天赋，所以就只好去网上「拿来主义」--依照别人的网页，做自己的设计 😊 。这篇文章是是记录这个改版过程。

    信息收集

    首先要做的一件事就是去网上看看别人的设计，收集起来才好做到集「百家所长」。经过一段时间的网上搜索，我整理了下面几个网站，可以在改版时借鉴。

    • 载入进度条 : 这个网站的载入进度条有特色，可以借鉴。
    • 鼠标单击特效、载入进度条 : 这个站的鼠标单击特效、进度条可借鉴。
    • 页面布局 : 此站布局可借鉴。
    • 阅读进度条、toc浮动 : 此站阅读进度条及浮动toc可借鉴。
    • 博客主题编写 ： 可能用到的主题编写技术
    • 炫酷站 : 评论页及站点的其它设置
    • 配色及布局 : 此站简洁布局，配色舒服
    • toc浮动 : toc可借鉴
    • Gridea主题 : Gridea主题
    • 阅读进度条 : 阅读进度条
    • 时尚布局 : 为以后给老婆主页改版做准备
    • 模版网站 ： 一个模版网站
    • 主页logo图标 : 很好看的logo方法，强烈借鉴
    • 主页上的个人介绍 ： 布局非常好看，可以借鉴到老婆的主页
    • 炫酷站 : 非常好看的个人介绍站
    • 免费网站psd模版 : 免费的网页psd模版
    • 炫酷站 ：非常好看的个人介绍站
    • css工具 ：可能用到的CSS工具
    • CSS书写规范 ： CSS书写规范
    • 最主要的布局参考 : 🎏 🎏 🎏 这个网站所使用的模板 Clover 是我最主要的参考对象。非常感谢原作者的付出。

    目标设定

    看了这么多的个人博客之后，我大体上了解了自己心里想要的样式，主要包含以下几个点：

    • 😝 简洁，不要有太多的复杂元素，像是大量动画，或者酷炫的视频背景。
    • 🎨 配色、字体优雅
    • 🌈 不单调，虽然前定位是简洁，但是要有一定的点缀，像是鼠标特效、鼠标图标
    • 👨‍💻 代码高亮
    • 📒 有文章分类菜单，方便快捷访问相关类别的文章 ( 😂 😂 这个要求在其它的博客框架下很容易实现，但是在Gridea下没有办法，因为Gridea根本就没有「Category」这个属性。)
    • 😆 有一个简单，但是好看的主页， 例如ideaclover和Weihan这两个站。
