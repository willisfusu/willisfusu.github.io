    我为什么换掉印象笔记

    2018-01-08 | 笔记 想法 evernote | 2 min read

    此文章发表于 1441 天前，请注意文章时效

    [2018-01-09]

    主要是针对印象笔记在 Android 端上的缺点的吐槽。

    从 2012 年开始用印象笔记，用了两年付费用户。笔记也有小千条了。但是最近折腾了一晚上，把印象笔记上的笔记导入到了为知笔记中。为什么对印象笔记不满呢？主要有下面的原因：

    1. Android 客户端上从其它软件分享来创建笔记这个功能基本上是废的。软件界面只显示 clipping…永远也不会剪好。有时候连标题都没有。这个问题有好几年了，到现在没解决。
    2. 通过其它方式的公众号内容，印象笔记不能剪。微信现在作为高质量阅读来源，和第三方的阅读工具相容性极差。腾讯公司简直恶意满满，手机上要装好多个软件用来阅读。阅读源不能一元化。这里同时批下第三方工具 pocket instapaper 只能看到图片，但是在分享到印象笔记时，就没有图片了。难道把图片格式转换下就那么难嘛？
    3. 印象笔记 chrome 端的剪藏工具不能剪微信公众号内容，没有图片。只能通过 firefox 端来剪。原因是 firefox 不支持 webp 格式的图片……简直扯淡呀。我是不可能放弃 chrome 的。因为试用了两个周 firefox，桌面版还好。安卓端比 chrome 慢了不止一点半点。桌面版也是各种蛋疼，比如卸载插件的麻烦程度……让我怀疑其产品经理脑子有问题。

    当然，为知是个小公司，刚上手 wiznote，就感觉到其和印象笔记的差距：

    1. 网页工具就是垃圾……真的。
    2. 同步速度让人崩溃……
    3. 和第三方软件的相互通信不好。pocket 不能直接将内容推送到为知笔记。但是可以直接推送到印象笔记。
    4. 小公司，不知道什么时候就完了，总是有些担心……
    5. 加标签的时候不能自动补全。

    现在我还在同时使用 Wiz 与 Evernote，也在考虑如何能将两者的优点结合起来。等到后面实验有结果了，我再来更新这个文章。
