    自用 Chrome 插件整理推荐

    2018-04-12 | 日常 chrome | 6 min read

    此文章发表于 1149 天前，请注意文章时效

    为什么写这篇推荐

    上个周在实验室里和师兄聊天的时候，发现师兄的 Chrome 竟然一个插件都没有装，大感意外，于是就给师兄推荐了几个自己一直在用的插件。在推荐的过程中，把一些插件的主要用途、应用场景及缺点都介绍了一遍。然后今天我就在想，是不是有必要把自己使用的 Chrome 插件整理一下，把一些主要的功能及应用方法记录下来，这样不管是推荐给其他人或者是自己需要的时候都可以很快的查到。所以就有了这篇文章。

    前期准备

    1. Chrome 浏览器 （必需）
    2. 插件安装包（非必需下载地址 密码：xyvk）/*此处需要补充链接地址*/

    ##推荐名单

    1. Evernote Web Clipper ⭐️
    2. IDM Integration Module
    3. Keepa （或者惠惠助手）
    4. Remebear
    5. Stylish
    6. Tampermonky ⭐️⭐️
    7. uBlock Origin ⭐️
    8. Zotero Connector ⭐️⭐️

    功能介绍

    Evernote Clipper

    这个插件是印象笔记的网页剪辑插件，想必在用印象笔记的朋友对这个插件一点也不陌生。使用起来非常的简单，在需要剪辑的页面点一下就可以自动保存到印象笔记中了。 保存的方式也有很多种，可以保存网页截图、鼠标选择部分、网页正文文章或者整个网页页面。并且在保存时，还会智能的匹配你印象笔记中的笔记本和标签。个人使用下来，感觉智能匹配的还是相当准确的。这样就不至于保存到印象笔记中之后，笔记条目乱的不行。
    其它其实也有不少的笔记应用支持网页剪辑，但是还没有见过印象笔记这么好用的。

    优点：使用方便，内容选取准确。
    缺点: 如果使用的是印象笔记的账号，但是在国外登录，那么就需要先将系统语言换为英文，然后才会有转到印象笔记的选项，不知道官方怎么想的，搞的这么麻烦。。。。。

    IDM

    其实这个插件放在这里不是为了说这个插件有多么的厉害。这个插件只是用来监测浏览器下载行为的，真正神器的是 IDM 软件本体。利用 IDM 搭配一个百度云盘的直链插件，可以轻轻松松实现百度网盘的快速下载。这里只是简单的提一下，后面如果有时间再补一个如何搭配 IDM 下载百度云盘的方法。

    Keepa

    Keepa 和惠惠都是用来监测跟踪电商产品价格的。这个插件相对而言还是相当有用的。他们可以提供某个产品在最近几个月内的价格变动。同时也可以提供不同平台同一产品的价格，对于买东西的时候比价还是相当有帮助的。每当黑五或者双十一的时候，就可以通过他们看到心愿单里的产品最近价格变动以及商家是不是真的打折，还是先提价再降价了。不过惠惠在双十一的时候有很大的可能会失效。。。但是 Keepa 又只能监测 Amazon 的产品。按需要选择吧。

    优点：提供产品某个时间段内的价格变动及不同平台的价格对比。
    缺点：Keepa 只支持 Amazon，惠惠可能在双十一时失效。。。。

    Stylish

    这个也是相当有名的插件，看名字就是和样式相关的插件。这个插件可以做什么呢？

    可以实现网页样式的自定义。换句话说，你可以自定义网页的样子，进行美化。

    什么地方用呢？ 很多地方都可以。我主要用在 google 及微博。微博真的是太可怕了，各种不相关的框框和广告。使用这个插件就可以美化他。去掉那些不相关的。还一个简洁的微博来。

    先来两个对比图， 是不是没有那些乱七八糟的东西了？

    使用方法也是非常的简单：首先点击 Stylish 图标，选择为此网站选择更多样式，在打开的网页中选择需要的样式就可以了。 当然，如果你熟悉 CSS，也可以自己编写针对网站的样式。

    Tampermonky 油猴插件

    这个插件可以说是最最神器的插件了，里面应用尽有，Doraemon 一样，可以满足你的各种要求。简单来说，这就是一个网页脚本的管理工具，具体的功能实现需要有对应的脚本。所以有两个选择：1 自己写脚本 2 使用别人的脚本。好在网上的能人不少，已经写了很多的脚本, 比如你可以在Greasyfork 里下载到绝大多数的脚本。

    所主要使用的脚本如下：

    1. Userscript+ :这个相当有用，强烈推荐添加。可以提示你当前页面可以安装哪些脚本，及实现的功能。
    2. 百度直链 ：和上面提到的 IDM 搭配，下载百度网盘中的内容，非常好用，速度相当快，可以达到百度网盘 Vip 用户的速度。
    3. 知乎免登录： 在看一些逼乎上的回答时，想要多看几个回答，必需要登录。但是我又没有逼乎的账号，使用这个脚本就可以不用登录，看全部的回答。
    4. 解除 B 站地域限制。国外看不了国内的视频，因为有版权地域限制，有了这个插件，我就可以开心的上 B 站啦~~

    当然，还有很多其它的，比如 YOUKU, 爱奇艺 VIP 视频播放等。有了第一个脚本，Tampermonky 会提醒你当前页面的可用插件的。

    uBlock

    广告与屏蔽广告之间的斗争一直在持续着。最近有没有感觉到 ADblock 有点乏力呢？可以试试 uBlock，相比 adblock 更加激进一些。 当然也可以自定义屏蔽广告的规则。

    Zotero

    这个绝对的也是神器，尤其是对于做科研工作的朋友。我之前一直使用 Noteexpress 与 EndNote 来管理文献。其实 NoteExpress 是要比 EndNote 要好用的，可惜我这边的学校只买了 Endnote,用了几个月，只能放弃，很多功能没有，比如标签，比如好用可搜索的笔记功能。可惜又用不了 Noteexpress,然后我就发现了 Zotero 这个软件。重新定义了我管理文献的方法。有兴趣的朋友可以了解下，后面有时间，会补一篇使用 Zotero 来管理文献的文章。
